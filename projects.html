Stock Price Simulator

provided by Finance and Economics Club,IIT GUWAHATI

Tech used: Python, yfinance, matplotlib, seaborn along with few others, monte carlo simulation

The project is about extracting financial data of a particular stock corresponding to a ticker symbol. Using this data historical returns are calculated. Using Scipy library a skewnorm distribution is generated. Now as is done in Monte Carlo Simulation distinct random walks(simulations) are generated for one month. Each of the random walk is plotted to give a tentative trend of future stock prices
